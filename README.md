# GraphQL Chat Server

## Setup

```bash
$ git clone git@github.com:yuttasakcom/graphql-chat-server.git
$ npm install
```

## Start Prisma

```bash
$ cd prisma && docker-compose up -d
$ prisma deploy --env-file ../dev.env && cd ..
```

## Get Schema

```bash
$ npm run get-schema
```

## Run Dev

```bash
$ npm run dev
```
